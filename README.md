# Hello fellow contributors!!
Here is the task for you. Good luck 👍.

This project is a simple web application that presents a zero-themed site. If you open the `/hello` path of the site, you will encounter an error. Fork this repository, fix the bug and submit a PR.

Once you do that, make sure you document the installation instructions, the fix you made as an edit to this file and push that as a separate commit with the message `"added documentation"`.

This resource will be useful for you: 
https://www.markdownguide.org/cheat-sheet/

Good luck :)
